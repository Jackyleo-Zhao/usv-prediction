# AIS-USV-Prediction

本仓库对应论文 **《基于残差增强物理模型的低频 AIS 无人艇短时轨迹预测方法》**，聚焦于**低频 AIS 条件下无人艇短时轨迹预测**这一工程问题。论文作者为 **赵明艳（Mingyan Zhao）**，本人为该工作的**第一作者**。

与直接围绕模型复杂化展开的常见做法不同，本研究首先关注低频工程 AIS 数据本身的可靠性。在当前数据条件下，AIS 位置报告中的动态信息与由相邻位置点计算得到的运动学参数并不总是严格一致，存在典型的“**字段—位移不一致**”现象。围绕这一问题，论文构建了一条完整的技术路径：

**一致性分析 → 统一基准建立 → 操纵性参数恢复与校准 → 残差增强物理预测 → 预测误差范围表达**

## 项目概述

低频 AIS 数据虽然具有覆盖范围广、获取成本低等优势，但在工程应用中，AIS 字段链路中的 SOG/COG 与由经纬度位移重建得到的对地航速、对地航向和对地航向变化率并不总是同步一致。若直接基于AIS 位置报告中的动态信息估计导数量，容易在短时机动条件下引入结构性偏差，并进一步放大为操纵性参数失真和轨迹外推误差。

因此，论文首先对 AIS 位置报告中的动态信息与由相邻位置点计算得到的运动学参数开展定量一致性分析，并将**由相邻位置点计算得到的运动学参数**确定为后续操纵性分析、模型构建与性能评估的统一运动学参考基准。

在统一基准基础上，论文进一步从AIS 位置报告中的动态信息与由相邻位置点计算得到的运动学参数对比中恢复对地航速、对地航向变化率、回转半径和减速距离指标的量级关系，形成面向低频 AIS 条件的操纵性参数恢复与工程校准思路；随后构建基于 **CV-CT 物理模型** 与 **MLP 残差修正** 的短时轨迹预测框架 **RAP-CT**，并基于同类预测条件下的历史终点误差统计结果引入**预测误差范围表达**，增强结果的工程表达能力。

## 核心内容

### 1. 一致性分析与统一运动学参考基准

论文首先回答是“当前低频 AIS 数据能否直接作为可靠的运动学建模依据”。研究表明，AIS 位置报告中的动态信息与由相邻位置点计算得到的运动学参数之间存在显著系统性偏差：速度一致性 MAE 为 **0.903 m/s**、RMSE 为 **0.907 m/s**，固定阈值下失配比例达到 **89.9%**，相对阈值下达到 **100%**，并存在明显时滞现象。基于这些证据，论文将由相邻位置点计算得到的运动学参数确定为统一工程参考基准，而将 AIS 位置报告中的动态信息保留为对照参考。

### 2. 操纵性参数恢复与工程校准

在统一参考基准建立之后，论文进一步比较了 AIS 位置报告中的动态信息与由相邻位置点计算得到的运动学参数在对地航速、对地航向变化率、回转参数和减速距离指标上的差异，指出AIS 位置报告中的动态信息在低频采样条件下对导数量存在明显平滑低估，并会经由非线性传播进一步放大为操纵性参数失真。基于此，论文提出了面向低频 AIS 条件的操纵性参数恢复与工程校准思路，使后续物理模型参数设置建立在更合理的数量级基础上。

### 3. 残差增强物理预测框架 RAP-CT

在上述基础上，论文构建了 RAP-CT 预测框架。该框架以恒速恒转率模型（CV-CT）作为物理模型，先生成中心预测轨迹，再利用多层感知机对**对地航速残差**与**对地航向变化率残差**进行联合校正，从而形成“**CV-CT 中心轨迹 - MLP 残差修正 - 预测误差范围表达**”的协同预测机制。该设计的重点在于在统一物理基准下，通过物理约束与局部残差补偿共同抑制误差累积。

### 4. 预测误差范围表达

除中心轨迹预测外，论文还基于同类预测条件下的历史终点误差统计结果构建了预测误差范围表达，用于表达误差随预测时域扩展的变化范围。其作用是在工程层面增强短时轨迹外推结果的结构化表达能力，使预测结果不再只是单条中心轨迹，而同时携带误差可能扩展的范围信息。

## 主要结果

在 2、10 和 20 min 预测任务中，RAP-CT 在总体测试窗口上的终点误差均值分别为 **14.00、63.27、110.98 m**，优于 CV-CT 的 **23.72、123.98、241.47 m**；路径均方根误差均值分别由 **13.67、71.42、143.74 m** 降至 **9.56、46.52、85.18 m**。

结果表明，RAP-CT 在保持物理可解释性的前提下，有效提高了“字段—位移不一致”低频 AIS 条件下无人艇短时轨迹预测的总体精度，并增强了预测结果的工程表达能力。

进一步地，代表性起点分析表明，该方法在 **P50 典型片段** 与 **P90 高误差片段** 中均具有稳定优势，且在困难场景和较长预测时域下改善更为明显。
